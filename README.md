# One Bit Health

Calculadora de IMC desenvolvida como prática do aprendizado em React Native durante o curso "React Native (aprendiz)" do canal One Bit Code no YouTube.
link para o canal: [One Bit Code](https://www.youtube.com/@OneBitCode)

## Descrição do App
A aplicação é uma calculadora de IMC desenvolvida para celulares Android o iSO. A pessoa que a usar deverá preencher campos com altura e peso e clicar
no botão para calcular. O resultado será mostrado na mesma tela juntamente da mensagem indicando qual a classificação do IMC calculado.

## Tecnologias utilizadas
Todo o projeto foi desenvolvido utilizando React Native e TypeScript.
